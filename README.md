# KKSK_MC_CVT
简单的小工具，能将网易版我的世界存档转换为国际版存档。支持自动修复存档玩家数据，支持导出为适用于手机端的.mcworld文件。图形化界面。  /   A simple little tool that can convert NetEase version Minecraft save files into international version save files. It supports automatically repairing player data in the save files and exporting them as .mcworld files suitable for mobile devices. It has a graphical interface.
